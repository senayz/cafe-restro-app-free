# Cafe POS APK Release

This release contains the Android APK build for Cafe POS.

## Installation

1. Download the APK file from this release.
2. Open the APK on your Android device.
3. If prompted, allow installation from unknown sources.
4. Install and open the app.

## Notes

- This release is APK only.
- No source code package is required for installation.
- Please back up your cafe data before updating from an older version.

## Support

For support, contact through the in-app Support button.
